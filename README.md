# TODO List

  This todo list application designed to keep and track your daily tasks and help your productivity. It is built using the Vite + React project template.

## Installation

  Run command 
  ```
  npm install
  ```
## Development

  ### Start the local development server:

    To run app
    ```
    npm run dev
    ```
    To stop app
    ```
    Ctrl + C
    ```
